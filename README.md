# Kamera-Footage-Organizer

Sortiert Dateien aus einem beliebigen Ordner in die dafür vorgesehenen Ordner:
- Kamerafotos: Hier landen alle Fotos, die unbearbeitet sind ; Für entrauschte Bilder gibt es hier einen Unterordner
- Bearbeitet: Hier landen alle Fotos, die bearbeitet sind
- Videos: Hier landen alle Videos
- Sonstige: Hier landet alles, was in diesem Ordner nichts zu suchen hätte
